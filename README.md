# Personal Finance Tracker

A finance tracker for anyone having difficulty handling and managing their finances and tracking a record with persistent data storage, and showing clear transaction data to help manage their finances seamlessly and have a record.

## Features

- Add transactions
- Edit transactions
- Delete transactions
- See Current Balance/Income/Expense
- Search (debounced)
- Filter by category
- Dark/Light mode
- Local Storage
- Import JSON
- Export JSON
- Responsive design

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Installation

1. Clone repository
```bash
git clone https://github.com/AribKhan218/Personal-Finance-Tracker
```
2. Open `index.html`
3. Start it with Live Server (VS Code extension)

## How It Works

1. Transactions stored in an array
2. Saved in Local Storage
3. UI rendered dynamically
4. Event delegation for edit/delete
5. Search and filter update rendered data
